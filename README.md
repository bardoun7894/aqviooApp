# Aqvioo – AI Video Generation App

Aqvioo is a Flutter mobile app (iOS/Android) that generates promotional videos and images from text ideas or uploaded photos. It integrates AI services for marketing copy, visuals, and voiceovers in Arabic and English, with authentication and payment flows.

## Overview
- Text-to-video and image-to-video generation
- AI-powered marketing copy and voiceover (Arabic/English)
- Firebase Auth (phone OTP), Firestore, Storage
- Payments (Tabby, Apple Pay, STC Pay)
- User dashboard and Admin web dashboard

## Prerequisites
- Flutter SDK installed (`flutter --version`)
- Dart SDK bundled with Flutter
- Configured platforms (Android Studio/Xcode as needed)

## Quick Start
1. Install dependencies: `flutter pub get`
2. Run the app: `flutter run`

## Project Structure
- `lib/main.dart` – App entry point
- `lib/` – Screens, widgets, services, models, utils, themes
- `docs/` – Project and API documentation
