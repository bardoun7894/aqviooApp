# Data Safety Section Answers
# إجابات قسم سلامة البيانات

Use these answers when completing the Google Play Console Data Safety section.

---

## Overview

The Data Safety section tells users what data your app collects and shares. Answer these questions based on Aqvioo's functionality.

---

## Data Collection

### Does your app collect or share any of the required user data types?
**Answer: Yes**

---

## Data Types Collected

### 1. Personal Info

#### Phone Number
- **Collected**: Yes
- **Shared**: No
- **Purpose**: Account management (Authentication via OTP)
- **Optional**: No (Required for login)
- **User can request deletion**: Yes

---

### 2. Financial Info

#### Purchase History
- **Collected**: Yes
- **Shared**: No (processed by Tap Payments)
- **Purpose**: App functionality (Credit purchases)
- **Optional**: Yes (only if user makes purchases)
- **User can request deletion**: Yes

---

### 3. Photos and Videos

#### Photos
- **Collected**: Yes
- **Shared**: Yes (with AI processing services)
- **Purpose**: App functionality (AI image/video generation)
- **Optional**: Yes (only if user uploads photos)
- **User can request deletion**: Yes

#### Videos
- **Collected**: Yes (generated content)
- **Shared**: No
- **Purpose**: App functionality
- **Optional**: N/A (generated by app)
- **User can request deletion**: Yes

---

### 4. App Activity

#### App Interactions
- **Collected**: Yes
- **Shared**: No
- **Purpose**: Analytics
- **Optional**: No
- **User can request deletion**: Yes

---

### 5. App Info and Performance

#### Crash Logs
- **Collected**: Yes
- **Shared**: No
- **Purpose**: Analytics (App stability)
- **Optional**: No
- **User can request deletion**: No

#### Diagnostics
- **Collected**: Yes
- **Shared**: No
- **Purpose**: Analytics
- **Optional**: No
- **User can request deletion**: No

---

### 6. Device or Other IDs

#### Device ID
- **Collected**: Yes
- **Shared**: No
- **Purpose**: Analytics, Fraud prevention
- **Optional**: No
- **User can request deletion**: No

## Security Practices

### Is data encrypted in transit?
**Answer: Yes**
- All data transmitted using HTTPS/TLS

### Can users request data deletion?
**Answer: Yes**
- Users can contact support@aqvioo.com to request account and data deletion

## العربية - ملخص البيانات المجمعة

### البيانات التي نجمعها:
1. **رقم الهاتف** - للتحقق من الهوية
2. **سجل المشتريات** - لإدارة الرصيد
3. **الصور** - التي يرفعها المستخدم للمعالجة
4. **الفيديوهات** - المحتوى المُنشأ
5. **تفاعلات التطبيق** - للتحليلات
6. **سجلات الأعطال** - لتحسين التطبيق
7. **معرف الجهاز** - للتحليلات

### البيانات التي نشاركها مع أطراف ثالثة:
- الصور والنصوص مع خدمات الذكاء الاصطناعي للمعالجة
- معلومات الدفع مع مزود الدفع

### ممارسات الأمان:
- جميع البيانات مشفرة أثناء النقل
- يمكن للمستخدمين طلب حذف بياناتهم
