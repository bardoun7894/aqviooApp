# Screenshot & Graphics Requirements
# متطلبات لقطات الشاشة والرسومات

---

## Required Assets for Google Play Store

### 1. App Icon (Required)
| Specification | Value |
|---------------|-------|
| Size | 512 x 512 px |
| Format | PNG (32-bit) |
| Max file size | 1 MB |
| Transparency | No |
| Notes | Should match your app's launcher icon |

---

### 2. Feature Graphic (Required)
| Specification | Value |
|---------------|-------|
| Size | 1024 x 500 px |
| Format | PNG or JPG |
| Max file size | 1 MB |
| Notes | Main promotional banner, shown at top of store listing |

**Suggested Design:**
- App logo/name prominently displayed
- Tagline: "Create AI Videos in Seconds" / "أنشئ فيديوهات بالذكاء الاصطناعي في ثوانٍ"
- Show sample AI-generated content
- Use your brand colors (purple/white glassmorphism theme)

---

### 3. Phone Screenshots (Required - Min 2, Max 8)
| Specification | Value |
|---------------|-------|
| Size | 1080 x 1920 px (9:16 portrait) |
| Format | PNG or JPG |
| Max file size | 8 MB each |
| Quantity | Minimum 2, recommended 4-6 |

**Recommended Screenshots (in order):**

1. **Home Screen / Creation Wizard**
   - EN: "Describe your idea and let AI create"
   - AR: "صف فكرتك ودع الذكاء الاصطناعي يبدع"

2. **Style Selection**
   - EN: "Choose from multiple AI models"
   - AR: "اختر من نماذج ذكاء اصطناعي متعددة"

3. **AI Generation Loading**
   - EN: "Watch your vision come to life"
   - AR: "شاهد رؤيتك تتحول إلى واقع"

4. **Preview Screen (Video Result)**
   - EN: "Professional results in minutes"
   - AR: "نتائج احترافية في دقائق"

5. **My Creations Gallery**
   - EN: "All your creations in one place"
   - AR: "جميع إبداعاتك في مكان واحد"

6. **Share Screen**
   - EN: "Share instantly to social media"
   - AR: "شارك فوراً على وسائل التواصل"

## Screenshot Creation Tips

### Tools to Use:
- **Figma** (Free) - https://figma.com
- **Canva** (Free) - https://canva.com
- **Screenshot frames** - https://screenshots.pro or https://mockuphone.com

### Best Practices:
1. Use device frames (iPhone/Android mockups)
2. Add promotional text overlays
3. Keep text minimal and readable
4. Show actual app screens (not conceptual)
5. Highlight key features
6. Use consistent branding/colors
7. Create separate sets for English and Arabic
