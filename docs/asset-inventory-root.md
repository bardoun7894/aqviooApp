# Asset Inventory

This document lists the static assets included in the application bundle.

## Fonts
**Package:** `google_fonts`
- The application uses Google Fonts dynamically, so font files are not stored locally in assets.
- **Primary Font:** Likely `Cairo` or `Tajawal` (for Arabic support) and `Outfit` or `Inter` (for English), based on design specs.

## Localization
**Directory:** `lib/l10n/`
- `app_en.arb`: English translations
- `app_ar.arb`: Arabic translations
